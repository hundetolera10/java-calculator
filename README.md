# Java Calculator

A simple graphical calculator built in Java using **Swing**.  
It supports basic arithmetic operations (+, -, *, /) and handles decimal numbers. Ideal for learning Java GUI and event-driven programming.

## Features
- Add, subtract, multiply, and divide numbers
- Handle decimal input properly
- User-friendly interface with buttons for digits and operations
- Prevents multiple decimal points in a single number
